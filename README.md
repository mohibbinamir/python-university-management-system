# Python University Management System

A Python academic project based on a University Management System. The system uses text files for storing and managing records related to students, lecturers, administrators, attendance, modules, grades, fees, and receipts.

## Project Overview

This project was developed to simulate a simple university management environment using Python and file handling. The system manages different user roles and their related records in a structured way.

## Features

- Record management using text files
- Student and lecturer data handling
- Attendance and grade record support
- Fee and receipt record handling
- Course and module information storage
- Role-based file organization

## Technologies Used

- Python
- File Handling
- Text-Based Data Storage
